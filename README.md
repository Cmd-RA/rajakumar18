# ChannelVista - Professional Photo Social Platform

ChannelVista is a high-performance, mobile-first social platform for visual storytellers. It features AI-powered captioning, real-time feeds, and a built-in monetization system.

## 🚀 Deployment Guide (GitHub to Hosting)

This project is built with **Next.js 15**, **Firebase**, and **Genkit AI**. To host this website:

### 1. Push to GitHub
- Create a new repository on GitHub.
- Push this code to your repository.

### 2. Connect to Firebase App Hosting (Recommended)
- Go to the [Firebase Console](https://console.firebase.google.com/).
- Navigate to **App Hosting**.
- Connect your GitHub repository.
- Firebase will automatically detect `apphosting.yaml` and the Next.js build settings.

### 3. Setup Environment Variables
In your hosting dashboard (Firebase or Vercel), add the following environment variable:
- `GEMINI_API_KEY`: Your Google AI API key for the caption and moderation features.

### 4. Google AdSense Verification
- Once the site is live, go to the **Admin Panel** (`rajahribabakumar@gmail.com`).
- Go to the **System/Ads** tab.
- Paste your Google AdSense verification code.
- Click **Save**. The code will be injected into all pages automatically.

## 🛠 Features
- **Real-time Home Feed:** Instant updates for new photos.
- **AI Tools:** Speech-to-text captions and content moderation.
- **Monetization Dashboard:** Tracks 5000 follower threshold.
- **Master Admin Panel:** Global control for the owner.
